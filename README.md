# ayvaroff.github.io

This project is a personal CV and portfolio website, built with TypeScript, Nunjucks, and Tailwind CSS. It features a modern, responsive design and includes sections for experience, projects, and education. The site is statically generated and deployed via GitHub Pages.

The design was originally created using [Stich](https://stitch.withgoogle.com/) Google AI and has been thoughtfully adapted to suit the needs of this project.

## Features

- Clean, responsive layout
- CV and project showcase
- Built with TypeScript, Nunjucks, and Tailwind CSS
- Easy to update and extend

## Getting Started

1. Clone the repository
2. Install dependencies with `yarn`
3. Build the site with `yarn build`
4. Open the `docs/` folder to view the generated site
